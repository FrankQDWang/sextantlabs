# 01. Writing Workbench

## 界面目标

主界面应该是写作工作台，不是聊天窗口。

```text
┌──────────────────────────────────────────────┐
│ Top Bar: Project / Chapter / POV / Mode       │
├───────────────┬──────────────────┬───────────┤
│ Memory Panel  │ Writing Editor   │ Agent     │
│ optional      │ main focus       │ Panel     │
├───────────────┴──────────────────┴───────────┤
│ Candidate Drawer                              │
└──────────────────────────────────────────────┘
```

## 区域职责

| 区域 | 职责 | 默认状态 |
|---|---|---|
| Writing Editor | 正文创作、选中、插入候选 | 永远可见 |
| Agent Panel | 发起动作、控制生成模式 | 可折叠 |
| Candidate Drawer | 展示候选、风险、采纳按钮 | 有候选时出现 |
| Memory Panel | 当前 POV、角色、证据、伏笔 | 默认收起 |
| Review Panel | 风险列表、未处理事项 | 默认收起或小徽标 |

## 不做

- 不用纯聊天框替代工作台；
- 不自动覆盖正文；
- 不强制用户先处理所有风险；
- 不把 Memory 做成数据库管理界面。
